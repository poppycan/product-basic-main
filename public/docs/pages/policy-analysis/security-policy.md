# 安全策略

## 功能概述

安全策略页面用于按设备（防火墙）维度检索、筛选与查看策略数据，支持按生命周期、策略质量、业务标签等维度统计与过滤，便于发现有问题策略并执行详情、路径分析、合规分析等操作。

## 核心需求

- **需求背景**：防火墙策略数量多、分布在不同设备，需要统一入口按设备与条件检索，并识别过期、停用、冗余、冲突等问题策略。
- **实现目标**：提供左侧设备树 + 右侧检索与列表的布局，支持多选设备、多种检索方式、标签统计过滤与行内操作。
- **解决问题**：集中查看多设备策略、快速定位问题策略、关联专线/备注/命中信息，支撑策略优化与回收流程。

## 目标用户

- 安全运维、网络管理员：日常策略巡检与问题排查。
- 策略申请人/负责人：查看策略状态、备注与命中情况。
- 合规/审计人员：按生命周期与质量标签做合规分析。

## 主要功能点

- 左侧设备过滤树：设备组与防火墙层级、多选防火墙、启停用标识、右键菜单（设备管理/详情/配置更新/下载/删除）。
- 右侧检索栏：普通检索与高级检索切换、含 any/不含 any、匹配模式、重置、常用检索保存与回填。
- 策略标签统计：生命周期状态、策略质量、业务标签，可折叠，点击过滤列表。
- 策略列表：策略信息、源/目的/服务、状态与优先级、NAT 信息、关联专线、策略备注、命中信息、标签及行内操作（详情、路径分析、合规分析、策略回收、去优化、编辑、删除、查看命令）。

## 使用说明

1. 在左侧树中勾选需要查看的防火墙，右侧列表将仅展示这些设备下的策略。
2. 在检索栏输入条件后点击「检索」或按 Enter，可配合「含 any」「匹配模式」缩小范围。
3. 将当前条件点击「设为常用检索」后可快速回填并检索。
4. 展开「策略标签统计」后，点击具体标签可过滤出对应类型的策略。
5. 列表中可对单条策略进行详情、路径分析、合规分析、编辑等操作；右键防火墙节点可进行设备管理、配置下载等。
