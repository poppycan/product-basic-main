# NAT 开通

> 本文档为 **业务向页面说明**（《页面 UI 规范》§2.1.1），供页头「页面说明」展示：只描述功能、规则与界面交互，不写路由、代码与接口细节。完整功能范围与验收见 **功能级 PRD**（NAT 开通）；研发对照见仓库内 **技术对照** 文档。

## 功能概述

在「策略变更」域内提供 **NAT 开通** 能力：运维人员在选定防火墙上录入 NAT 开通意图，查看列表中的执行与校验情况，并通过弹窗完成新增或编辑。本页帮助用户**集中查看与筛选** NAT 开通单，并进入表单维护。

**非目标（本页不单独承担）**

- 不替代底层设备上的真实配置生效过程；实际下发、回撤与回读以平台与设备对接为准。
- 设备资产来源以纳管体系为准；本页仅提供可选的设备筛选入口。

## 元素与功能点

### 设备树

- **作用**：选中一台设备后，列表仅关注该设备相关开通单（语义以产品为准）。
- **未选**：展示符合其它条件的全部相关条目（若产品要求必须选设备，以 PRD 为准）。

### 时间与状态

- **时间**：未选时间通常表示不按创建时间限制；选定后仅保留创建时间落在区间内的条目。
- **状态色块**：在**当前时间条件**（若已选）基础上，再按「聚合业务状态」过滤；与列表展示一致。

### 查询条件

- **主查询**：模式（如 SNAT/DNAT 等）、策略相关标识、VR、描述等，与其它条件为 **「同时满足」**关系。
- **更多查询**：行状态、源/目的安全域与地址、服务、出入接口、BNAT 相关、安全域、NAT 类型、策略位置参考、执行状态等；全部与主查询 **同时生效**。
- **查询 / 重置**：提交组合条件刷新列表；重置则清空条件并按产品规则恢复默认。

### 列表与行操作

- **列**：展示设备、模式与类型、地址与转换、VR、执行状态与时间、描述等；长文案以界面规则截断，悬停可看完整说明（若已实现）。
- **添加**：打开 **NAT 开通** 弹窗，新建一条开通单。
- **编辑**：打开同弹窗；是否允许改表单取决于该行**当前执行状态**（见弹窗说明）。
- **导出 / 删除**（若展示）：按产品确认范围；删除前应有确认，避免误删。

### 页头「页面说明」

- 打开后展示本文，帮助用户与协作方对齐**本页能做什么、条件怎么组合**。

## 输入原则与规范

- **组合查询**：各筛选项之间为 **AND**；空条件通常表示「不限定该维度」。
- **时间**：自定义时需同时有起止意义；快捷项含义与展示格式以界面为准。
- **设备**：单选；切换设备后应重新理解列表为「该设备维度下的开通单」。
- **分页**：翻页只影响浏览位置，不改变已选筛选条件（除非产品另有「查询后重置到第一页」等规则）。

## UI 交互与状态

### 主路径

1. 进入 NAT 开通页 → （可选）选设备 → 设时间、状态、查询条件 → **查询** → 浏览列表 → **添加** 或 **编辑** → 在弹窗内保存后回到列表并看到更新（联调后以实际为准）。

### 空态与加载

- 无数据：提示无记录或可引导调整条件/添加（文案以界面为准）。
- 加载中：列表区域有加载反馈，避免重复点击提交。

### 执行与校验（业务理解）

- 列表可展示**执行状态**（如待下发、成功、失败、回撤等）及时间；若需**人工校验**，界面可用图标提示，点击后表示已完成人工确认（具体以产品规则为准）。

### 与其它策略变更入口

- 页头子入口可在各子功能页间切换，不隐藏其它入口，避免迷路。

## 子功能说明（弹窗）

- **添加 / 编辑 NAT 开通**：从列表 **添加** 或行上 **编辑** 进入；详细字段、输入规则与按钮含义见 **同模块弹窗业务说明**（弹窗内「原型说明」加载的文档）。
