## 一、产品概述

### 1.1 功能定位

安全策略页面是策略分析模块的核心页面，用于按设备（防火墙）维度检索、筛选与查看策略数据。页面支持按生命周期、策略质量、业务标签等多维度统计与过滤，便于发现有问题策略并执行详情、路径分析、合规分析等操作。

### 1.2 需求背景

- 防火墙策略数量多、分布在不同设备，需要统一入口按设备与条件检索
- 需识别过期、停用、冗余、冲突等问题策略
- 需集中查看多设备策略、快速定位问题策略
- 需关联专线/备注/命中信息，支撑策略优化与回收流程

### 1.3 目标用户

| 角色 | 使用场景 |
|------|----------|
| 安全运维、网络管理员 | 日常策略巡检与问题排查 |
| 策略申请人/负责人 | 查看策略状态、备注与命中情况 |
| 合规/审计人员 | 按生命周期与质量标签做合规分析 |

## 三、功能模块详述

### 3.1 左侧设备过滤树

#### 3.1.1 树结构

| 层级 | 说明 |
|------|------|

#### 3.1.3 交互功能

| 功能 | 说明 |

| 设备搜索 | 支持按设备名称或 IP 关键字过滤树节点 |
| 多选设备 | 勾选防火墙节点作为右侧列表的 `deviceIds` 过滤条件 |
| 添加设备 | 点击「+」图标添加新设备（占位） |

#### 3.1.4 右键菜单（仅防火墙节点）

| 菜单项 | 说明 |
|--------|------|
| 设备管理 | 跳转设备管理页面（URL 占位） |
| 设备详情 | 弹出设备详情弹窗（占位） |

### 3.2 右侧检索区

#### 3.2.1 检索模式

| 模式 | 说明 |
|------|------|
| 普通检索 | 单行输入，空格或逗号分隔：策略名称/ID、动作、源 IP、目的 IP、端口 |
| 高级检索 | 命令式编辑器，支持字段:值、操作符（=、!=、in、nin、>=、<=）、连接词（and、or、end）、子表达式（括号） |

#### 3.2.2 高级检索字段（22 个）

- **网络类**：源 IP、源区域、目的 IP、目的区域、端口、协议
- **NAT 类**：转换前源 IP、转换后源 IP、转换前目的 IP、转换后目的 IP、转换前端口、转换后端口
- **策略类**：优先级、动作（允许/拒绝）
- **档案类**：关联专线、负责人、申请人、执行人、业务、备注、注释、工单

#### 3.2.3 检索条件

| 条件 | 说明 | 默认值 |
|------|------|--------|
| 含 any | 多选：含 any、不含 any | 无选中 |
| 匹配模式 | 单选：全包含、包含、排除、相等、部分匹配 | 全包含 |
| 标签过滤 | 通过策略标签统计区点击添加 | 空 |

#### 3.2.4 操作按钮

| 按钮 | 说明 |
|------|------|
| 检索 | 触发搜索，支持 Enter 快捷 |
| 重置 | 清空所有条件（含 any、匹配模式、标签过滤、高级表达式） |

### 3.3 常用检索

| 功能 | 说明 |
|------|------|
| 设为常用 | 将当前检索条件保存为常用检索（localStorage，key: `securityPolicy.commonSearches`，最多 5 条） |

## 七、附录

### 7.1 生命周期标签定义

- **已过期**：引用时间对象已过有效期
- **已停用**：策略状态为已停用
- **5天内到期**：还有 5 天即将过期
- **30天内到期**：还有 30 天即将过期
- **档案缺失**：档案信息为空的策略

### 7.2 策略质量标签定义

- **冗余**：小范围允许策略被高优先级大范围允许策略完全包含
- **冲突**：源 IP、目的 IP、服务、区域相同，动作不同
- **隐藏拒绝**：小范围拒绝策略被高优先级大范围允许策略完全包含
- **重复**：源/目的域、源/目的 IP、服务、动作都完全相同
- **空区域**：源区域或目的区域为空
- **空对象**：引用对象为空

### 7.3 风险与合规标签定义

- **矩阵访问违规**：不符合合规矩阵访问规则
- **高危端口违规**：含高危端口
- **含 any 策略**：源/目的/服务任意包含 any
- **宽泛策略**：IP 或端口数量超过阈值
- **日志未启用**：未启用日志
- **备注为空**：无备注信息
